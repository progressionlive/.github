# progressionlive/.github

Organization-level GitHub configuration for all Progression repositories.

## Starter Workflows

Templates available when creating new workflows via **Actions → New workflow** in any org repo.

| Template | Description |
|----------|-------------|
| `slack-pr-notify` | Posts a Slack notification to `#code-reviews` when a non-draft PR is opened or marked ready |

## Community Health Files

Default files (e.g., `CONTRIBUTING.md`, `CODE_OF_CONDUCT.md`) placed at the root apply to all org repos unless overridden.

## Org Profile

`profile/README.md` renders on the [org's GitHub page](https://github.com/progressionlive).
